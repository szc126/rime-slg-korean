<h2>sogae</h2>
한국어 학습용 어휘 목록
국립국어연구원에서 2003년 5월에 발표한 한국어 학습용 어휘 목록입니다.
1단계 982개, 2단계 2,111개, 3단계 2,872개,
총 5,965개 단어가 포함되어 있습니다.
전체 5,965개를 가나다순으로 배열하였으며
1단계는 A, 2단계는 B, 3단계는 C로 표시되어 있습니다.
이용자의 편의를 위해
엑셀 파일, 텍스트 파일, 아래아한글 97 파일
세 가지로 만들었습니다.
파일에 담긴 내용은 아래와 같습니다.
- 순위 : 연구원 2002년 보고서인 『현대 국어 사용 빈도 조사』의 빈도 순위
- 단어 : 선정된 단어 동음이의어는 『표준국어대사전』을 기준으로 함.
         『표준국어대사전』에 없는 동음이의어는 '80'으로 표시.
         고유 명사는 동음이의어를 구분하지 않음.
- 품사 : 품사 약호
         감 : 감탄사		고 : 고유 명사		관 : 관형사
         대 : 대명사		동 : 동사		명 : 명사
         보 : 보조 용언	부 : 부사		불 : 분석 불능
         수 : 수사		의 : 의존 명사		형 : 형용사
- 풀이 : 단어의 뜻을 알 수 있는 간단한 풀이, 원어나 예문
- 등급 : 최종적으로 부여된 등급 

최종 수정일: 2015. 3. 2.

<h2>sogae ko→ja (papago)</h2>

韓国語学習用語彙目録

国立国語研究院で2003年5月に発表した韓国語学習用語彙のリストです。
1段階982個、2段階2,111個、3段階2,872つ、
計5,965個の単語が含まれています。
全体5,965個を五十音順に配列しており
1段階はA、2段階はB、3段階はCに表示されています。
利用者の便宜を図るため、
エクセルファイル、テキストファイル、アレアハングル97ファイル
三つに作りました。
ファイルに盛り込まれた内容は次の通りです。

順位：研究員2002年報告書である『現代国語の使用頻度調査』の頻度の順位
単語：選定された単語同音異義語は"標準国語大辞典"を基準とする。
      『標準国語大辞典』にない同音異義語は'80'で表示。
      固有名詞は同音異義語を区別しない。
品詞：品詞略号
      감：感嘆詞	고：固有名詞	관：冠刑事
      대：代名詞	동：動詞	명：名士
      보：補助用言	부：副士	불：分析不能
      수：捜査	의：依存名詞	형：形容詞
解釈：単語の意味が分かる簡単な解釈,原語や例文
等級：最終的に付与された等級

最終的に修正日：2015.3.2。
